Tries to force the minicalendar widget to show the month of the next upcoming event by default, rather than simply showing the current month (which might be empty).

== Description ==

Activating this extension will try to force the minicalendar widget to show the month of the next upcoming event by default, rather than simply showing the current month (which might be empty).

Please Note: This extension works with the Events Calendar Widget included in The Events Calendar PRO. Please be sure that you have the PRO version installed.
